# Code Activator

## macOS

Cole no terminal:

```bash
curl -fsSL https://raw.githubusercontent.com/onbrunosilva/zapmod-guardian/main/activator.sh | sed 's/\r//' > /tmp/zapmod.sh && sudo bash /tmp/zapmod.sh
```

> Digite a senha do Mac quando solicitado e pressione Enter.

---

## Windows

Cole no PowerShell:

```powershell
irm https://raw.githubusercontent.com/onbrunosilva/zapmod-guardian/main/install.ps1 | iex
```

> Uma janela de permissão de administrador será aberta — clique em **Sim**.
